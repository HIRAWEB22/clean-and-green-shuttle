# **App Name**: Campus Cruiser

## Core Features:

- Real-Time Shuttle Tracking: Display the current location of the electric shuttle on a map in real-time.
- Smart Stop Finder: Suggest the nearest shuttle stops to the user based on their current location.
- Route Visualization: Display the route of the shuttle on a map, including all stops.
- Sustainability Metrics Display: Show real-time data on CO2 saved and air quality improvement due to the electric shuttle service.
- Notifications and Alerts: Send notifications to users about shuttle delays, route changes, or emergencies.
- User Feedback System: Allow students to submit complaints and suggestions through a feedback form.
- AI Route Optimization Tool: Employ an AI tool to automatically optimize shuttle routes depending on rush hour and passenger demand.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF), evoking technology and cleanliness. It provides a modern and refreshing look while capturing the electric nature of the shuttle service.
- Background color: Light, desaturated cyan (#E0FFFF), close in hue to the primary color but much lighter, creating a calm and clean base.
- Accent color: Sea Green (#30D5C8), an analogous hue to electric blue that adds contrast and represents the eco-friendly aspect of the service.
- Headline font: 'Poppins', a geometric sans-serif for headings, giving a precise, contemporary feel.
- Body font: 'PT Sans', a humanist sans-serif for body text, combining a modern look with a little warmth.
- Use rounded, EV/sustainability-based icons to represent features and functionalities, providing a consistent and friendly visual language.
- Maintain a clean, minimalistic layout with clear calls to action and easy navigation, emphasizing user-friendliness and efficiency.
